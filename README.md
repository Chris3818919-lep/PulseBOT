# 🤖 PulseBOT

PulseBOT is a powerful Discord analytics bot built with Python and SQLite.

Track server activity, discover your most active members, monitor channel engagement, and gain insights into your community—all through modern Discord slash commands.

---

## ✨ Features

### 📊 Analytics
- Track every message sent in the server
- View total server activity
- Monitor active users and channels

### 🏆 Leaderboards
- View the most active members
- See which channels generate the most activity

### 👤 User Statistics
- Check message counts for individual members
- View user information instantly

### 👋 Welcome System
- Automatically welcome new members
- Beautiful embedded welcome messages

### 💾 Data Storage
- Powered by SQLite
- Persistent server analytics
- Fast and lightweight

---

## 🛠 Commands

| Command | Description |
|----------|-------------|
| `/dashboard` | View Pulse dashboard |
| `/stats` | Show server statistics |
| `/userinfo` | View a user's stats |
| `/leaderboard` | Show server leaderboard |
| `/topusers` | Most active users |
| `/topchannels` | Most active channels |
| `/serverinfo` | Information about the server |
| `/about` | Information about PulseBOT |
